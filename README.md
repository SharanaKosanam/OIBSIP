# OIBSIP
Oasis Infobyte Python Internship Projects

## About the Internship
The Oasis Infobyte Python Programming Internship is a 4-week internship program designed to improve Python programming skills through practical project development. The internship focuses on hands-on Python projects, problem-solving, real-world learning, resume building, and technical growth. To successfully complete the internship, at least **three projects** must be completed and submitted.

## Projects Included

### 1. Password Generator
This project generates a random password based on the length entered by the user.

**Features:**
- Takes password length as input
- Uses letters, digits, and special characters
- Generates a random strong password

**Concepts Used:**
- `random` module
- `string` module
- User input
- String handling

---

### 2. BMI Calculator
This project calculates the **Body Mass Index (BMI)** using the user's weight and height.

**Features:**
- Takes weight in kilograms as input
- Takes height in centimeters as input
- Converts height from cm to meters
- Calculates BMI
- Displays BMI category:
  - Underweight
  - Normal weight
  - Overweight
  - Obese

**Concepts Used:**
- Arithmetic operations
- Conditional statements
- User input
- Type conversion

---

### 3. To-Do List
This project is a simple command-line To-Do List application used to manage daily tasks.

**Features:**
- Add new tasks
- View all tasks
- Remove tasks
- Exit the application

**Concepts Used:**
- Lists
- Loops
- Conditional statements
- User interaction

---

### 4. Quiz Application
This project is a basic quiz program that asks the user questions and checks their answers.

**Features:**
- Displays quiz questions
- Accepts answers from the user
- Checks whether answers are correct
- Calculates and shows final score

**Concepts Used:**
- Conditional statements
- Variables
- User input
- Simple scoring logic
