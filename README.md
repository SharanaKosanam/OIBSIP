# OIBSIP
Oasis Infobyte Python Internship Projects

This repository contains the Python projects completed by me as part of the **Oasis Infobyte Python Programming Internship**.

## About the Internship
The Oasis Infobyte Python Programming Internship is a 4-week internship program designed to improve Python programming skills through practical project development.  
The internship focuses on:
- Hands-on Python projects
- Problem-solving skills
- Open-source collaboration
- Resume building
- Real-world project experience

To successfully complete the internship, at least **three projects** must be completed and submitted.

## Projects Included

### 1. Password Generator
This project generates a strong random password based on the length entered by the user.

**Features:**
- Takes password length as input
- Uses letters, numbers, and special characters
- Generates a random secure password

**Concepts Used:**
- `random` module
- `string` module
- User input
- Loops and string handling

---

### 2. BMI Calculator
This project calculates the **Body Mass Index (BMI)** using the user's weight and height.

**Features:**
- Takes weight in kilograms as input
- Takes height in centimeters as input
- Converts height from cm to meters
- Calculates BMI
- Displays BMI category:
  - Underweight
  - Normal weight
  - Overweight
  - Obese

**Concepts Used:**
- Arithmetic operations
- Conditional statements
- User input
- Type conversion

---

### 3. To-Do List
This project is a simple command-line To-Do List application that helps users manage daily tasks.

**Features:**
- Add new tasks
- View all tasks
- Remove tasks
- Exit the application

**Concepts Used:**
- Lists
- Loops
- Conditional statements
- User input
